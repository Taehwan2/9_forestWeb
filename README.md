# 숲웹🏕️

## 프로젝트 
### 개요
- "스마트 배달 제한 시스템"은 사용자의 건강을 보호하기 위해 일정 횟수 이상의 배달음식 주문이 이루어질 경우 자동으로 차단하는 웹 서비스입니다.
- 사용자가 설정한 주문 횟수 한도에 도달하면, 시스템은 자동으로 배달 음식 주문을 제한하며, 대신 건강한 식단 선택에 도움이 될 수 있는 정보를 제공합니다.
- 내가 사용한 소비내역 입력을 통한 지출계획을 한눈에 보는 서비스 구축
